Coin Flip Game
A simple web game that simulates flipping a coin using HTML, CSS, and JavaScript.

 Technologies Used
- HTML
- CSS
- JavaScript

 Features
- Click button to flip a coin
- Random output: Heads or Tails
- Clean, minimal interface
